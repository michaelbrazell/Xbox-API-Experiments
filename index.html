Xbox API Experimentation

I am building an application for interacting with Xbox Live data.

About

Currently, all of the data is specific just to my XBL username, MichaelBrazell, but I will be adding user-specific data in time. There are some key limitations to the Xbox API, largely that you only have 120 requests/hour with a standard developer key, but that can be expanded higher if need be. For that reason, this will be an on-demand service, e.g., that you would request a data refresh and be rate-limited to X per hour or per day, per user.
